# Source IP Analysis

## Objective
Analyze VPN source IP addresses to identify suspicious or high-volume connection activity.

---

## Detection Logic
Monitor source IP addresses generating:
- Excessive VPN connections
- Repeated authentication attempts
- Connections from unexpected regions

---

## Investigation Focus
- Most active IP addresses
- Repeated connection attempts
- Geographic anomalies
- Suspicious traffic patterns

---

## Example Fields Used
- Source_ip
- Source_Country
- UserName

---

## Why It Matters
Attackers often use repeated connection attempts or suspicious IP addresses to gain unauthorized access.

Source IP analysis helps identify abnormal activity patterns.
